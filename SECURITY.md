# Sécurité

Ne signalez pas publiquement une vulnérabilité exploitable. Contactez l'équipe
responsable du dépôt avec l'impact, les étapes de reproduction et une proposition
de correction. Aucun secret réel ne doit être stocké dans Git ou dans une image.

Les versions supportées et le canal privé de signalement seront publiés avant la bêta.
